# csv-load-and-process

## Contents

This node module can read csv and process them.

return Two-dimensional array of String type.

Made by DoLonaAnd.

---
## Usage

```js
// import
const getCSV = require('csv-load-and-process');

// use
let path = "./text.csv",
    csvArray = getCSV(path);

/*
 * csvArray = [
 *  [csv,csv], //line
   [csv,csv,csv], //line
 *  [csv]  //line
 * ];
 */

```
